# Project instructions for Codex

- Build environment is MSYS2 UCRT64 and build system is makefile.
- Keep C code C99-compatible.
- Avoid large refactoring unless needed for the requested change.
- Preserve existing public APIs.

- Add a short header comment line to each function of C and Python codes.
- Minimize the comments. If needed, 1 line as maximum.
- Avoid #ifdef hook only for unit tests.
